# Student Grade Calculator

## Project Description

This is a Python-based Student Grade Calculator that processes multiple students' marks, calculates averages, assigns grades, and displays class statistics.

## Features

- Accepts data for multiple students
- Calculates average marks
- Assigns grades automatically
- Provides performance comments
- Validates user input
- Displays results summary
- Shows class statistics
  - Total Students
  - Class Average
  - Highest Average
  - Lowest Average

## What I Learned

- Loops (for and while)
- Lists and Dictionaries
- Input validation using try-except
- Calculating statistics using built-in functions

## Grading System

| Grade | Marks Range | Comment |
|---------|------------|----------|
| A | 90 - 100 | Excellent! |
| B | 80 - 89 | Very Good! |
| C | 70 - 79 | Good! |
| D | 60 - 69 | Needs Improvement! |
| F | Below 60 | Failed! |

## How to Run

```bash
python grade_calculator.py
```
